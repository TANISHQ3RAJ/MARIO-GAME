# Mario Game

Play the live game here: [https://mariobytan.netlify.app/](https://mariobytan.netlify.app/)

## 📖 Overview

A web-based Mario Game built with modern web technologies. This project features a full-stack architecture with a React frontend powered by the Phaser game engine and an Express backend.

## 🚀 Technologies Used

### Frontend
- **React** (via Vite)
- **Phaser** (Game Engine)
- **Tailwind CSS** (Styling)
- **Supabase** (Database / Auth)
- **TypeScript**

### Backend
- **Node.js** & **Express**
- **TypeScript**

## 📂 Project Structure

- `/frontend` - Contains the React and Phaser game code.
- `/backend` - Contains the Express server code.

## 🛠️ Getting Started

### Prerequisites
- Node.js installed

### Installation & Setup

1. **Backend Setup**:
   ```bash
   cd backend
   npm install
   # Start the backend server (if dev script exists, or configure appropriately)
   ```

2. **Frontend Setup**:
   ```bash
   cd frontend
   npm install
   # Start the development server
   npm run dev
   ```
